# Icons Directory

Place your application icon here.

## Required Files

**Minimum (required):**
- `icon.png` - Main application icon (recommended: 64x64 or 128x128 pixels)

**Optional (for better quality):**
- `icon.svg` - Vector version (scalable, best quality)
- `icon-256.png` - High resolution version (256x256 pixels)

## Specifications

- **Format:** PNG (with transparency) or SVG
- **Recommended sizes:** 64x64, 128x128, or 256x256 pixels
- **Background:** Transparent
- **Style:** Simple, recognizable design that works at small sizes
